## API Reference

#### Get all books

```http
  GET /api/books 
```



#### Create book

```http
  POST /api/books
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `title`   | `string` | **Required**. titulo del libro |
| `author`  | `string` | **Required**. author del libro  |
| `isbn`   | `string` | **Required**. codigo del libro |
| `genre`   | `string` | **Required**. genero del libro |
